什么是FreeMarker?

FreeMarker是一款模板引擎：即一种基于模板、用来生成输出文本(任何来自于HTML格式的文本用来自动生成源代码)的通用工具。它是为Java程序员提供的一个开发包，或者说是一个类库。他不是面向最终用户的，而是为程序员提供的一款可以嵌入他们所开发铲平的应用程序。
